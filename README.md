## Uruchomienie z Docker Compose
Polecenia:
- Uruchom (zbuduj, jeśli potrzeba):
  ```
  docker-compose up --build
  ```
  lub (nowa składnia):
  ```
  docker compose up --build
  ```
- Uruchom w tle:
  ```
  docker-compose up -d
  ```
- Zatrzymaj i usuń kontenery:
  ```
  docker-compose down
  ```
